# U-Net
A cheap, shameless copy and modernization of the original U-Net from 2015.

## Overview of the main files:
- See ***carvana.ipynb*** for the neural network's outputs on the Carvana dataset.
- Use ***train.ipynb*** to train your own model based on the U-Net architecture.
- ***dataset.ipynb*** depicts the various methods used to transform and normalize the images in the dataset.
- ***visualization.ipynb*** includes the methods used to visually analyze the neural network's outputs.
- ***weights.ipynb*** depicts outputs from the methods used to implemented weighted emphasis on the edges of objects.
